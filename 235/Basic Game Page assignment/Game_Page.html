Clair Obscur: Expedition 33

Publishing Info

  • Published by: Sandfall Interactive
  • Developed by: Sandfall Interactive
  • Released: 2025
  • Platforms: PlayStation 5, Xbox Series X|S, Windows

Description

Clair Obscur: Expedition 33 is a turn-based role-playing game set in a hauntingly beautiful fantasy world. Players lead an expedition of adventurers determined to stop the deadly Paintress, whose yearly cycle erases a number — and countless lives along with it. Featuring tactical combat, deep storytelling, and a painterly art style, the game merges classic RPG elements with modern innovation.

Game Categories

  • Genre: Role-Playing (RPG)
  • Perspective: Third-person
  • Gameplay: Turn-based combat, Tactical strategy
  • Setting: Fantasy
